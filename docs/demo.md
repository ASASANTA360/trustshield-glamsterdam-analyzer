# TrustShield AI Demo Script

This script is designed for a concise 2-minute grant review video.

## Tested contracts

- USDC on Ethereum: `0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48`
- WETH on Ethereum: `0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2`

## 2-minute walkthrough

### 0:00–0:20 — Problem and value proposition

Open the homepage and introduce TrustShield AI as a smart contract security and Glamsterdam-readiness analyzer for Ethereum grant reviewers, builders, and protocol teams. Point out the hero section, supported networks, and reviewer-friendly positioning.

### 0:20–0:45 — Easy scan flow

Scroll to the contract scanner. Click the USDC example button to prefill the Ethereum address. Explain that reviewers can paste any supported contract address or use canonical examples for a fast demo.

### 0:45–1:20 — Result dashboard

Run the scan and show the Trust Score, Security Grade, and Risk Level cards. Explain that raw JSON has been converted into clear decision-support cards and that the same report remains available through the REST API.

### 1:20–1:45 — Findings and recommendations

Scroll through the Contract Overview, Findings table, and Recommendations section. Emphasize bytecode size, evidence, severity, and concrete next actions for a contract owner or reviewer.

### 1:45–2:00 — Ecosystem relevance

Close on the Supported Networks section and mention Ethereum, Base, Polygon, and Arbitrum coverage. Reiterate that TrustShield AI helps the Ethereum ecosystem evaluate Glamsterdam readiness with repeatable, multi-chain evidence.

## Optional alternate take

Repeat the scan with WETH on Ethereum to demonstrate that the scan flow works with another canonical, high-recognition contract.
